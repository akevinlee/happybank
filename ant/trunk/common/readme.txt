Please note, the files in this directory should normally be placed in
another directory/component rather than alongside the project. They are 
placed here simply to aid in getting a working Ant project.

Contents of this directory includes:

 - common-macros.xml           - a set of common macros for Ant
 - common-targets.xml          - a set of common targets for Ant
 - default.properties          - default properties for all projects
 - tomcatTasks.properties      - definitions of Apache Tomcat Ant tasks
 - ivysettings.xml             - default settings for Apache Ivy
 - ivy.xml                     - library dependencies for common build tasks
 - lib\			               - common project libraries (only Ivy)
 - repository\                 - local Apache Ivy repository
